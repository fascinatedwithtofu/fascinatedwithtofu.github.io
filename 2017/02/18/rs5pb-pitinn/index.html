【ライブレポート】RS5pb @ 新宿 Pit Inn 2017/02/07

おなじみ 新宿ピットイン にて類家心平氏率いる PS5pb (Ruike Shinpei 5 piece band) やっと観れました。

メンバー

RS5pb
類家心平（Tp）田中”tak”拓也（G）中嶋錠二（P,Key）鉄井孝司（B）吉岡大輔（Ds）

わりと普通の編成に見える

過去のライブレポを読んでおりエキセントリックかなと構えておりました。が、意外と VJ もなく通常営業に見えました。＊演奏はぶっ飛んでいます。

セットリスト

怠けてすみません。近くにメモしている紳士をお見かけしました。どうぞお見かけしましたら教えてください。
基本的に UNDA のものが多かったです。

1st set …..
2ndset …..
En cole - Maisha / Miles Davis

所感

最近さらに露出も増え、先日は水谷浩章 3DAYSでもリップサービス vol.9でも表情の豊かさを大いに楽しませてにもらいましたが、一番好みの演奏が聴けたのがこの RS5pb です。
とにかく間隙を入れずにブロウし続ける。エフェクターやマイクなどは鳴らしてからがなんぼやと言っているかのような激しい音圧で始終圧倒されぱなしでした。
電化マイルスを正当に受け継いだようなプレイですが、新鮮味しかありません！！

個人的に面白いなと思った点は、ギター田中氏のロックに歪んだサウンドの’違和感’と、ドラム吉岡氏のちょうどよい合いの手＆チルアウトの合図の妙でした。異次元ソロでも安心してのれました。

さいごに

類家氏の使われているマイクがトランペットのフォルムと馴染んでいて、しかも激しいプレイにも耐えられる装着型で印象に残りました。もしやと思い調べたところおそらく SD Systems 社がマイルスと共同でデザインした LCM77 for Trumpet だと思います。先っぽ黒いカバーを外した状態で使われていますね。
